# Plan Command Readiness Evidence

This directory records implementation evidence for `fsgg-sdd plan`.

Expected evidence files:

- `build-release.txt`: Release build output.
- `artifact-plan-tests.txt`: focused artifact-model plan parser and id tests.
- `command-plan-tests.txt`: focused command plan tests.
- `plan-create-tests.txt`: create-path command evidence.
- `plan-rerun-tests.txt`: rerun, safe-addition, deferral, and stale-decision evidence.
- `plan-blocked-tests.txt`: blocked prerequisite and unsafe existing-plan evidence.
- `plan-output-tests.txt`: deterministic JSON, text projection, dry-run, and Governance-boundary evidence.
- `full-suite.txt`: full Release test suite output.
- `fsi-plan-surface.txt`: public FSI/prelude command surface evidence.
- `cli-plan-json-smoke.txt`: disposable-project JSON CLI smoke output.
- `cli-plan-dry-run.txt`: disposable-project dry-run CLI smoke output.
- `cli-plan-text-smoke.txt`: disposable-project text CLI smoke output.
- `performance.txt`: local plan create and rerun timing evidence.
- `sdd-governance-boundary-review.md`: SDD/Governance boundary review.
- `human-summary-review.md`: text projection review.
- `artifact-traceability.md`: source, contract, implementation, and evidence traceability.
