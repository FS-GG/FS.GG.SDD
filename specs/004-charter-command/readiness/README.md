# Charter Command Readiness Evidence

This directory records real evidence for the `fsgg-sdd charter` feature.

Expected evidence files:

- `build.txt` - Release build output.
- `charter-create-tests.txt` - focused charter create and real interpreter tests.
- `charter-rerun-tests.txt` - safe rerun, safe addition, identity mismatch, and unsafe overwrite tests.
- `generated-view-tests.txt` - generated work-model state and refresh tests.
- `charter-diagnostics-tests.txt` - invalid request and duplicate work-id tests.
- `traceable-output-tests.txt` - deterministic JSON, text projection, and Governance boundary tests.
- `command-workflow-tests.txt` - public MVU boundary tests.
- `fsi-charter-public-surface.txt` - FSI prelude output through the public command surface.
- `charter-cli-smoke.txt` - disposable-directory CLI smoke output.
- `full-test-suite.txt` - full solution test output.
- `governance-boundary-review.txt` - SDD/Governance ownership review.
- `artifact-traceability.txt` - final task-to-artifact traceability.
- `performance.txt` - local command harness timing for create and rerun scenarios.
- `human-summary-review.txt` - review of text output against SC-006.
